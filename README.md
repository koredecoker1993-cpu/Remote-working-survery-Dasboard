# Remote Working Survey (RWS) — Data Analytics Project( Interactive Dashboard Creation using Ms Excel and Power bi)
# # Project Objective 
To analyse the Remote Working Survey (RWS) datasets from 2020 and 2021 in order to understand how COVID-19 impacted remote working patterns, productivity, time use, and workplace barriers among employees. The project aims to deliver data driven insights through structured KPI analysis and interactive Power BI dashboards, supporting organisations in making informed decisions about hybrid and remote working models in the post-pandemic era.
